Горячие клавишы Visual Studio Code

Основные

  • F1 – Показать палитру команд
  • CTR LSHIFT P – Показать палитру команд
  • CTRL P – Быстрое открытие, перейти к файлу
  • CTRL SHIFT N – Новое окно / экземпляр
  • CTRL SHIFT W – Закрыть окно / экземпляр

Редактирование

  • CTRL X – Вырезать строку (пустое выделение)
  • CTRL C – Копировать строку (пустое выделение)
  • CTRL [ – Уменьшить отступ
  • CTRL ] – Увеличить отступ
  • HOME – Перейти в начало строки

Навигация

  • CTRL T – Показать все символы
  • CTRL G – Перейти к строке
  • CTRL P – Перейти к файлу
  • ATL → – Перейти вперед
  • ATL ← – Перейти назад
